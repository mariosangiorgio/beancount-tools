# Usage
```
usage: amex.py [-h] [-c CATEGORY_MAP] [-d DATA]

Converts a csv file downloaded from American Express to a list of beancount
entries.

optional arguments:
  -h, --help            show this help message and exit
  -c CATEGORY_MAP, --category_map CATEGORY_MAP
                        The category map file
  -d DATA, --data DATA  The ofx.csv file
```
